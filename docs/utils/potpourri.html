fannypack.utils.squeeze(x: Any, axis: Union[int, Tuple[int, …]] = None) → Any¶

Generic squeeze function.

Parameters

  • x (Any) – Object to squeeze. Must have a shape attribute and be indexable with slices.

  • axis (Union[int, Tuple[int, ..]], optional) – Axis or axes to squeeze along. If None (default), squeezes all dimensions with value 1.

Returns

Any – Squeeze object.

fannypack.utils.to_device(x: Union[Any, torch.Tensor], device: torch.device, detach: bool = False) → Union[Any, torch.Tensor]¶

Move a torch tensor, list of tensors, dict, or dataclass of tensors to a different device.

Parameters

  • x – (torch.Tensor, list, tuple, dict, or dataclass) Tensor or container of tensors to move.

  • device (torch.device) – Target device.

  • detach (bool, optional) – If set, detaches tensors after moving. Defaults to False.

Returns

torch.Tensor, list, tuple, dict, or dataclass – Output, type will mirror input.

fannypack.utils.to_torch(x, device='cpu', convert_doubles_to_floats=True)¶

Converts a numpy array, list of numpy arrays, dict, or dataclass of numpy arrays for use in PyTorch.

Parameters

  • x – (np.ndarray, list, tuple, dict, or dataclass) Array or container of arrays to convert to torch tensors.

  • device (torch.device, optional) – Torch device to create tensors on. Defaults to "cpu".

  • convert_doubles_to_floats (bool, optional) – If set, converts 64-bit floats to 32-bit. Defaults to True.

Returns

torch.Tensor, list, tuple, dict, or dataclass – Output, type will mirror input.
